# Wheat Sustainability Analytics

## 📌 Problem
Inefficient use of agricultural resources affecting wheat productivity.

## 🎯 Objective
Optimize wheat yield while improving water and fertilizer efficiency.

## 📊 Dataset
This dataset includes 50 agricultural trials covering:
- Wheat cultivars
- Fertilizer treatments
- Irrigation levels
- Yield performance

## 📈 Dashboard
![Dashboard](images/dashboard.png)

## 🔍 Key Insights
- Nubaria 2 achieved the highest yield
- Late sowing window improved productivity
- Fertilizer selection significantly impacts yield
